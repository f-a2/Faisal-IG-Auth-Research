## 👋 About This Project / حول المشروع
**Faisal-IG-Auth-Research** is a sophisticated security research tool designed to analyze and test web authentication flows. It specifically explores how modern platforms like Instagram handle CSRF tokens, session persistence, and automated request filtering.

## 🤯 OMG! Features / المميزات
- **Dynamic Token Extraction:** Automatically scrapes CSRF tokens to bypass basic security blocks.
- **ASCII Dashboard:** A professional terminal-style logo for an elite user experience.
- **Rate-Limit Intelligence:** Detects server warnings and automatically waits to protect your IP.
- **Result Logging:** Saves successful hits into a separate file automatically.

## 🚀 Installation & Usage / طريقة الاستخدام

### Prerequisites / المتطلبات
- **Python 3.x**
- **Requests Library:**
  ```bash
  pip install requests
  ```

### Running / التشغيل
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/f-a2/Faisal-IG-Auth-Research.git](https://github.com/f-a2/Faisal-IG-Auth-Research.git)
   ```
2. **Navigate to folder:**
   ```bash
   cd Faisal-IG-Auth-Research
   ```
3. **Start the script:**
   ```bash
   python3 faisal.py
   ```
